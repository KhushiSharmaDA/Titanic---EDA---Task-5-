# Titanic Dataset – Exploratory Data Analysis

## Objective
Perform Exploratory Data Analysis (EDA) to identify patterns, trends, and relationships affecting passenger survival.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Female passengers had a higher survival rate than males.
- Higher class passengers were more likely to survive.
- Fare showed positive correlation with survival.
- Missing values were observed in Age and Cabin columns.

## Files Included
- titanic_eda.ipynb – EDA notebook
- Titanic_EDA_Report.pdf – PDF report
- train.csv – Dataset
